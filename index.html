IoT Multi-Sensor

Sensor which posts data to a server

Using an ESP8266 microchip, I created a custom Internet of Things Multi-Sensor.This sensor is capable of detecting motion, temperature, humidity, luminance, and carbon monoxide. The chip connects to the network via Wi-Fi and uses MQTT to post to a MQTT Server running on my desktop. Some effeciency measures are also implemented, such as polling the sensors after certain intervals. The programming is done primarily in C++ using Arduino IDE.

Home Automation

Using a python based H.A. server

Currently using an open source python based home automation server to automate my parent's home. Home Assistant is a wonderful tool to get started with bridging the gaps between current day home automation hubs/devices. I have configured our setup at home to meet the needs of the family; this consists of monitoring of temperature, motion, state of devices in order to perform the appropriate action, such as sending a notification. This setup even allows control via Google Home and Amazon Echo. I have utilized linux to stand up the server and python to configure this platform.
